# OneShop
#### An e-commerce site build in MERN stack 

### To run in dev env
1. clone the repo
2. cd to frontend and run "npm install"
3. cd back to root and run "npm install"
4. create a .env file containing : NODE_ENV, PORT, MONGO_URI, JWT_SECRET
5. run "npm run dev"
